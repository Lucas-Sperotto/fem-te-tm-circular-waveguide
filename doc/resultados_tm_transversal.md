# Resultados – Modos TM (Campo Transversal)
Este documento apresenta os gráficos vetoriais dos modos TM, representando o campo elétrico transversal ($E_x$, $E_y$) obtido a partir da componente $E_z$.
As setas indicam a direção e intensidade relativa dos vetores, com coloração baseada na magnitude.
